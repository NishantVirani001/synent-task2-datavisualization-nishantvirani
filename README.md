# Task 2: Data Visualization

## Problem Statement
Create meaningful visualizations using the Iris dataset to understand patterns and relationships between features.

## Dataset Details
Dataset: Iris Dataset

The dataset contains measurements of iris flowers from three species.

## Approach
- Loaded the dataset.
- Created bar charts.
- Generated histograms.
- Created scatter plots.
- Compared different features.
- Interpreted the visualizations.

## Results
- Species distribution was visualized.
- Feature relationships were identified.
- Data patterns became easier to understand.
- Meaningful insights were extracted.
